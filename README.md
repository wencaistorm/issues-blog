## 背景
[GitHub](https://github.com)作为全球最大同性交友网站，已经有人在其上面做了很多匪夷所思的事情，利用 `issues` 功能写博客就是其中之一……

## 简介
利用 `issues` 写博客有以下不便：
+ 无法自定义样式
+ 不利于 SEO
+ ……

## What is this?
利用 [GitHub](https://github.com) 提供的 [API](https://developer.github.com/v3/issues/#list-issues-for-a-repository) 获取 `issues` 内容，然后自定义样式
